# Typescript React Redux Webpack Sass Starter

* Typescript React Redux Webpack Sass Starter
* Typescript
* Redux state container
* Sass preprocessor
* Webpack module bundler
* Testing with Jest

## Available Scripts
```shell
npm run start
```
NOTE: The App will start in localhost:3000 using webpack dev server

```shell
npm run build
```

```shell
npm run test
```
